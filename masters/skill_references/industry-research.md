# 行业研究 Skill

> "先看数据，再看格局，最后问——这个行业在哪个位置？"

## 核心原则

**数据优先，框架驱动，可验证可追溯。**

- 所有结论必须有数据支撑或可追溯来源，能用数据的地方绝不用猜测
- 八大模块框架保证分析完整性，不遗漏关键维度
- 数据不可得的维度标注"数据暂缺"而非主观填充
- 本 Skill 不包含任何投资建议或主观判断，只呈现数据和事实

---

## 工作流总览

```
用户指定行业
    │
    ├─ Step 1: 行业定位与数据采集
    │   ├─ 确认行业分类标准（申万/证监会/自定义）
    │   ├─ 调用 AKShare 获取基础数据
    │   └─ 补充搜索行业最新动态

    ├─ Step 2: 核心分析（按八大模块）
    │   ├─ ① 行业概览
    │   ├─ ② 市场规模与增长
    │   ├─ ③ 产业链结构

    │   ├─ ⑦ 趋势与驱动力
    │   └─ ⑧ 风险识别
    │
    └─ Step 3: 综合报告输出
        ├─ 结构化 Markdown 报告
        └─ （可选）导出为文件
```

### 1.2 数据采集流程

**⚠️ 必须使用工具获取真实数据，不可凭记忆编造。**

#### A. 行业基础数据（AKShare）

```python
# 安装依赖（如未安装）
# pip install akshare pandas numpy

import akshare as ak

# ① 申万一级行业信息（含当前 PE/PB/股息率）
sw_info = ak.sw_index_first_info()

# ② 行业历史行情（用于估值分位计算）
# 示例：电子行业 801080
hist = ak.index_level_one_hist_sw(symbol="801080")

# ③ 行业成分股列表
stocks = ak.sw_index_cons(index_code="801080")

# ④ 东方财富行业板块（含涨跌幅/资金流向）
east_industry = ak.stock_board_industry_name_em()
```

#### B. 补充信息（web_search）

对于 AKShare 无法覆盖的定性信息（政策动态、技术路线、行业新闻），调用 `web_search` 补充：
- `"[行业名] 行业政策 2026"`
- `"[行业名] 市场规模 增速 2025 2026"`
- `"[行业名] 竞争格局 龙头 市场份额"`

### 1.3 数据处理规范

- **日期**：分析截止日为当前系统日期
- **历史区间**：默认取近5年数据计算分位，特殊行业（如周期性极强者）可扩展至10年
- **异常值处理**：PE为负或超过200倍的成分股在计算行业均值时应剔除或标记
- **交叉验证**：AKShare 数据与 web_search 获取的券商研报数据应交叉比对，重大偏差（>20%）需标注

## Step 2: 核心分析（八大模块）

### 模块① 行业概览

**目标**：用200字说清这个行业到底做什么的。

**必含要素**：
- 行业定义与边界（什么是这个行业，什么不是）
- 行业生命周期阶段（导入期/成长期/成熟期/衰退期）及判断依据
- 行业核心特征（高壁垒/重资产/技术驱动/政策敏感/周期性强等）

**输出模板**：
```
## 1. 行业概览
**行业定义**：[一句话定义]
**当前阶段**：[阶段名称]，判断依据：[渗透率/增速/集中度等数据]
**核心特征**：
- [特征1]：[数据支撑]
- [特征2]：[数据支撑]
```

### 模块② 市场规模与增长

**目标**：量化这个行业的"体量"和"速度"。

**必含要素**：
- 近3-5年市场规模与 YoY 增速
- 细分赛道规模占比及增速对比
- 未来2-3年增速预测及核心假设

**数据来源优先级**：
1. AKShare 行业指数历史行情 → 可推导行业市值变化趋势
2. web_search 获取券商/咨询机构已发布的规模数据
3. 行业协会/政府统计公报

**输出模板**：
```
## 2. 市场规模与增长
| 年份 | 市场规模 | 同比增速 | Source |
|------|---------|---------|--------|
| 2023 | XXXX亿 | - | ... |
| 2024 | XXXX亿 | XX% | ... |
| 2025 | XXXX亿 | XX% | ... |
| 2026E | XXXX亿 | XX% | ... |

**细分赛道**：
| 细分领域 | 规模占比 | 增速 | 增长驱动 |
|---------|---------|------|---------|
| ... | ... | ... | ... |
```

### 模块③ 产业链结构

**目标**：画出上中下游的价值地图。

**必含要素**：
- 产业链全貌（上游→中游→下游→终端）
- 各环节价值占比（饼图数据或百分比估算）
- 各环节议价能力判断（强/中/弱）及依据
- 关键瓶颈环节和利润集中环节

**输出模板**：
```
## 3. 产业链结构
**产业链全貌**：
上游：[原材料/设备] → 中游：[制造/加工] → 下游：[应用/销售] → 终端：[消费者/企业]

**价值分布**：
| 环节 | 代表企业 | 毛利率区间 | 价值占比 | 议价能力 |
|------|---------|-----------|---------|---------|
| 上游 | ... | XX%-XX% | XX% | 强/中/弱 |
| 中游 | ... | XX%-XX% | XX% | 强/中/弱 |
| 下游 | ... | XX%-XX% | XX% | 强/中/弱 |

**关键发现**：[利润最厚的环节在哪？瓶颈在哪？]
```

### 模块④ 竞争格局

**目标**：说清楚谁在抢肉吃，壁垒有多高。

**必含要素**：
- 市场集中度（CR4/CR5/CR8）及趋势
- 竞争类型判断（完全竞争/垄断竞争/寡头垄断/垄断）
- 龙头公司对比表（营收、市占率、核心优势、战略方向）
- 波特五力简析（现有竞争者/潜在进入者/替代品/供应商/购买者）
- 进入壁垒评估（资金/技术/政策/渠道/规模效应）

**输出模板**：
```

**市场集中度**：CR5 = XX%, CR10 = XX% → [集中/分散]格局，近3年[提升/下降/稳定]

**龙头对比**：
| 公司 | 营收(亿) | 市占率 | ROE | 核心壁垒 | 战略方向 |
|------|---------|-------|-----|---------|---------|
| ... | ... | ... | ... | ... | ... |

**波特五力**：
- 现有竞争：[激烈/温和]，原因：[...]
- 进入壁垒：[高/中/低]，核心壁垒：[...]
- 替代品威胁：[高/中/低]，来源：[...]
- 供应商议价力：[强/弱]，原因：[...]
- 购买者议价力：[强/弱]，原因：[...]
```

### 模块⑤ 财务指标对标

**目标**：这个行业的公司到底赚不赚钱、质量如何。

**必含要素**：
- 行业平均 ROE / ROA / 毛利率 / 净利率（近3年）
- 行业平均资产负债率 / 流动比率
- 行业平均营收增速 / 净利润增速
- 龙头 vs 行业平均对比（识别龙头溢价）

**数据来源**：从 AKShare 成分股数据汇总计算，或从 web_search 获取券商统计值。

**输出模板**：
```
## 5. 财务指标对标
| 指标 | 行业平均(2023) | 行业平均(2024) | 行业平均(2025) | 龙头A | 趋势 |
|------|-------------|-------------|-------------|-------|------|
| ROE | XX% | XX% | XX% | XX% | ↑/↓/→ |
| 毛利率 | XX% | XX% | XX% | XX% | |
| 净利率 | XX% | XX% | XX% | XX% | |
| 营收增速 | XX% | XX% | XX% | XX% | |
| 资产负债率 | XX% | XX% | XX% | XX% | |
| 经营现金流/营收 | XX% | XX% | XX% | XX% | |

**关键发现**：[盈利能力趋势？现金流质量？龙头溢价幅度？]
```

### 模块⑥ 估值分析

**目标**：行业现在是贵还是便宜。

**必含要素**：
- 当前 PE-TTM / PB 及近5年历史分位
- 估值温度计判断（低估/合理/高估/极端）及对应分位区间
- 行业间横向估值对比（该行业 vs 全市场 vs 相关行业）
- 估值驱动因素拆解（盈利驱动 vs 估值扩张）

**估值分位标准**：
| 分位区间 | 判断 | 含义 |
|---------|------|------|
| <20% | 显著低估 | 历史上只有不到20%的时间比现在更便宜 |
| 20%-40% | 偏低 | 处于历史中低区域 |
| 40%-60% | 合理 | 处于历史中枢附近 |
| 60%-80% | 偏高 | 处于历史中高区域 |
| >80% | 显著高估 | 历史上只有不到20%的时间比现在更贵 |

**输出模板**：
```
## 6. 估值分析
**当前估值**：
| 指标 | 当前值 | 5年历史分位 | 判断 |
|------|-------|-----------|------|
| PE-TTM | XX.X | XX% | 低估/合理/高估 |
| PB | X.X | XX% | |

**行业横向对比**：
| 行业 | PE-TTM | PB | ROE | 分位 |
|------|--------|-----|-----|------|
| 本行业 | ... | ... | ... | ... |
| 全市场均值 | ... | ... | ... | ... |
| 对标行业A | ... | ... | ... | ... |

**估值驱动分析**：[当前估值是由盈利支撑还是情绪驱动？]
```

### 模块⑦ 趋势与驱动力

**目标**：未来往哪走，什么在推。

**必含要素**：
- **短期驱动（1-2年）**：政策落地/产品周期/库存周期
- **中长期驱动（3-5年）**：技术变革/人口结构/消费升级/产业转移
- 关键催化剂事件和潜在拐点信号
- 未来3年行业增速的情景预测（乐观/中性/悲观）

### 模块⑧ 风险识别

**目标**：什么可能让你亏钱。

**必含要素**：
- **政策风险**：监管方向变化、补贴退坡、牌照收紧
- **技术风险**：技术路线颠覆、研发失败、专利纠纷

- **周期风险**：宏观周期位置、库存周期阶段
- 每个风险标注：概率（高/中/低）× 影响（高/中/低）= 风险等级

## 行业特殊处理规则

### 周期行业（钢铁/煤炭/有色/化工/航运等）
- 估值看 PB 而非 PE（盈利波动大时 PE 失真）
- 必须标注库存周期位置（补库/去库）
- 重点分析供需缺口，而非长期增长趋势

### 成长行业（AI/新能源/创新药/半导体等）
- 估值优先用 PEG 或终局 TAM 倒推
- 重点分析渗透率 S 曲线位置
- 研发投入/专利数/技术路线是关键指标

### 金融行业（银行/券商/保险）
- 估值看 PB/ROE 杜邦拆解
- 重点分析净息差/不良率/偿付能力充足率等行业特有指标
- 宏观利率环境是最强外部变量

### 消费行业（食品饮料/家电/纺织服饰等）
- 品牌护城河和渠道力比财务指标更重要
- 关注人口结构变迁和消费升级/降级趋势
- 龙头溢价逻辑需要定量验证

### 政策敏感行业（房地产/教育/医疗）
- 政策分析权重提升至 30%+
- 必须标注政策时间线和影响路径
- 估值需考虑政策风险折价

## 质量检查清单

报告完成前逐条检查：

- [ ] 所有数据点均有 Source 标注
- [ ] PE/增速等核心指标有交叉验证
- [ ] 竞争格局的 CR5/CR10 数据来源可追溯
- [ ] 估值分位基于至少 3 年历史数据
- [ ] 风险识别覆盖政策/技术/需求/竞争/周期五类
- [ ] 行业生命周期判断有数据支撑（渗透率/增速/集中度）
- [ ] 产业链价值分布有毛利率或利润占比数据
- [ ] 如果行业数据缺失 >50%，主动降级为「行业速览」而非「行业研究报告」

### 与投资大师 Skills 的关系

本 Skill 是纯数据驱动的行业研究工具。如需定性视角增强，你可以独立调用以下 Skills：
- `charlie-munger-perspective`：护城河审查、逆向思考、管理层激励评估
- `howard-marks-perspective`：周期定位、情绪温度、安全边际判断
- `cathie-wood-perspective`：技术颠覆 S 曲线、平台融合、终局 TAM

**建议使用方式**：先由本 Skill 输出完整数据报告，你再根据报告中的估值分位、技术特征等
自主判断是否调用大师 Skills 做定性补充。两者完全独立，大师视角的结论与数据呈现的结论
可能冲突——此时同时呈现，由你自行判断。
