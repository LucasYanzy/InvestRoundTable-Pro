## 4 心智模型（6个）

### 模型1: 文明现代化铁律
一句话：现代化3.0 = 自由市场 × 现代科技，这个组合不可阻挡。

思考路径：
1. 判断所在地区是否在现代化3.0的轨道上
2. 识别现代化进程中的受益者
3. 评估可持续性（制度/文化/技术协同）
4. 在趋势中寻找最确定标的

证据：比亚迪（押注中国制造业升级）、邮储银行（押注金融体系现代化）

局限：地缘政治割裂全球市场；技术速度超制度适应能力；文化与现代化发生根本冲突

### 模型2: 价值投资普适性
一句话：价值投资是普适的"大道"，在中国与美国一样适用。

思考路径：股票即所有权 → 市场先生提供的是服务 → 安全边际是核心技能 → 能力圈是前提

证据：在中国市场成功实践价值投资（比亚迪、邮储银行），哥大讲授价值投资课程

局限：市场长期非理性；公司治理不透明；投资者情绪失控

### 模型3: 能力圈
一句话：没有边界的能力不是真能力。

四层自知：知己所知 / 知己所不知 / 知己所不必知 / 知己所无知

证据：长期重仓少数公司，不追逐热点

局限：能力圈过窄错失机会；误判边界；不更新导致经验失效

### 模型4: 安全边际
一句话：安全边际是投资的核心技能，不是可有可无的选项。

思考路径：估算内在价值 → 计算价格差距 → 判断是否足够（30-50%）→ 不够就等

证据：2008年抄底美国银行、比亚迪投资时估值极低

局限：基本面恶化；找不到足够安全边际；估算系统性偏差

### 模型5: 市场先生
一句话：市场先生提供的是服务，不是指导。

思考路径：报价可以拒绝 → 市场情绪波动是机会 → 利用而非被影响 → 悲观时买入乐观时卖出

证据：2008年金融危机逆势买入，2020年疫情保持冷静

局限：市场长期"正确"；投资者无法控制情绪；报价机制被操纵

### 模型6: 理性诚实
一句话：理性诚实是最重要的投资品性。

四重诚实：对事实诚实 / 对逻辑诚实 / 对错误诚实 / 对无知诚实

证据：公开承认做空是投资生涯最严重错误，比亚迪持有20年

局限：利益冲突；群体压力；自我认知偏差
